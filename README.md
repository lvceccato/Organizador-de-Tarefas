# Organizador de Tarefas

Uma aplicação web desenvolvida com Flask para organizar tarefas, priorizar compromissos, acompanhar a produtividade por meio de um dashboard e exportar os dados para CSV, tudo em uma interface moderna e responsiva.

## O que faz

- Criar e organizar tarefas com datas de entrega
- Definir categorias personalizadas
- Gerenciar três níveis de prioridade (Muito Urgente, Urgente e Normal)
- Ordenar automaticamente as tarefas por data
- Visualizar apenas tarefas urgentes
- Acompanhar tarefas atrasadas, do dia e futuras em um dashboard
- Exibir a distribuição das tarefas por categoria
- Calcular automaticamente a taxa de atraso
- Exportar todas as tarefas para um arquivo CSV
- Disponibilizar uma API REST utilizando Flask

## Instalação

```bash
git clone https://github.com/lvceccato/Organizador-de-Tarefas.git

cd Organizador-de-Tarefas

pip install -r requirements.txt

python app.py
```

## Como usar

Abra o navegador e acesse:

```
http://127.0.0.1:9090
```

Depois basta adicionar tarefas, definir categorias e prioridades, acompanhar o dashboard e exportar sua lista de tarefas para CSV.
